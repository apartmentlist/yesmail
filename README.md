# Yesmail

A RubyGem to interact with the Yesmail API (http://www.yesmail.com/)

## Installation

Add this line to your application's Gemfile:

    gem 'yesmail'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install yesmail
